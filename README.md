# Telegram EMA Bot

A Telegram bot that calculates Exponential Moving Average (EMA) indicators for financial markets.

## Features
- Real-time EMA calculations
- Multiple time period support
- User-friendly interface

## Setup
1. Clone this repository:
   ```bash
   git clone https://github.com/darkdevil247/telegram-ema-bot.git
   cd telegram-ema-bot
   Run npm install

Set BOT_TOKEN in .env file

Run node bot.js
