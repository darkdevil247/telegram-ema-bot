# Telegram EMA Bot

A Telegram bot that calculates and provides Exponential Moving Average (EMA) indicators for financial markets.

## Features

- Calculate EMA for various time periods
- Support for multiple financial instruments
- Real-time or historical data analysis
- User-friendly Telegram interface

## Setup

### Prerequisites

- Node.js (version 14 or higher)
- Telegram Bot Token from [BotFather](https://t.me/BotFather)
- Railway account for deployment (optional)
